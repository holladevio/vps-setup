# vps-setup

## Update OS

```shell
sudo apt update && sudo apt upgrade
```

## Setup Server

```shell
sh -c "$(curl -fsSL https://raw.githubusercontent.com/holladevio/vps-setup/main/setup.sh)"
```
or
```shell
sh -c "$(wget -O - https://raw.githubusercontent.com/holladevio/vps-setup/main/setup.sh)"
```
